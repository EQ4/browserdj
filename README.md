# BrowserDJ

DJ Mixer for a web browser. 

This is an offline application. You can play DJ without uploading a music.

## Installation

	npm install -g browserdj

## Getting Started

1. Execute a command.

		browserdj

2. Go to a [http://localhost:3000](http://localhost:3000).
